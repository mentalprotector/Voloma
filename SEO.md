## Что нужно сделать для активации

### 1. Регистрация в Яндекс.Вебмастере
1. Перейдите на https://webmaster.yandex.ru/
2. Добавьте сайт: `https://voloma.94.140.224.220.sslip.io`
3. Получите код верификации
4. Добавьте в `src/app/layout.tsx`:
   ```typescript
   verification: {
     yandex: "ВАШ_КОД_ВЕРИФИКАЦИИ",
   },
   ```

### 2. Настройка Яндекс.Метрики
1. Перейдите на https://metrika.yandex.ru/
2. Создайте счётчик для сайта
3. Получите ID счётчика
4. Создайте файл `.env.local`:
   ```
   NEXT_PUBLIC_YANDEX_METRICA_ID=ВАШ_ID_СЧЁТЧИКА
   ```
5. В настройках счётчика включите:
   - ✅ Вебвизор
   - ✅ Карта кликов
   - ✅ Отслеживание событий

### 3. Настройка целей в Яндекс.Метрике
Создайте JavaScript-цели с идентификаторами:
- `hero_cta_click` — клик по CTA на главном экране
- `configurator_open` — открытие конфигуратора
- `option_shape_select` — выбор формы
- `option_size_select` — выбор размера
- `option_color_select` — выбор пропитки
- `option_quality_select` — выбор качества
- `gallery_image_view` — просмотр изображения
- `contact_click` — клик по контактам
- `custom_order_click` — индивидуальный заказ
- `submit_request` — отправка заявки

### 4. Регистрация в Google Search Console
1. Перейдите на https://search.google.com/search-console
2. Добавьте сайт
3. Получите код верификации
4. Добавьте в `src/app/layout.tsx`:
   ```typescript
   verification: {
     yandex: "...",
     google: "ВАШ_GOOGLE_КОД",
   },
   ```

### 5. Отправка sitemap.xml
После регистрации:
- **Яндекс.Вебмастер**: Индексирование → Карты сайтов → Добавить `/sitemap.xml`
- **Google Search Console**: Карты сайта → Добавить `/sitemap.xml`

## Проверка SEO

### Автоматические инструменты
1. **Lighthouse** (в Chrome DevTools):
   - Откройте главную страницу
   - F12 → Lighthouse → Generate report
   - Ожидаемый результат: 90+ по всем метрикам

2. **PageSpeed Insights**:
   - https://pagespeed.web.dev/
   - Введите URL сайта

3. **Rich Results Test** (Google):
   - https://search.google.com/test/rich-results
   - Проверьте корректность JSON-LD

### Ручная проверка
1. **Поисковая выдача** — через 1-2 недели после индексации:
   - Запрос: "деревянные кашпо для интерьера"
   - Запрос: "кашпо из карельской сосны"
   - Запрос: "кашпо Волома"

2. **Социальные сети** — проверьте превью при шаринге:
   - Facebook Sharing Debugger: https://developers.facebook.com/tools/debug/
   - Twitter Card Validator: https://cards-dev.twitter.com/validator

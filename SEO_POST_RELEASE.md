# Post-Release SEO План — Волома

> Выполнять после релиза на основной домен

## Этап 1: Регистрация в поисковых системах (Сразу после релиза)

### 1.1 Яндекс.Вебмастер
**URL:** https://webmaster.yandex.ru/

**Шаги:**
1. Зайти через Яндекс-аккаунт (или создать)
2. Нажать «+» → Добавить сайт
3. Ввести URL основного домена (например, `https://voloma.ru`)
4. Подтвердить права:
   - **Вариант А:** HTML-файл → скачать → положить в `public/yandex_[код].html`
   - **Вариант Б:** META-тег → скопировать код → добавить в `src/app/layout.tsx`:
     ```typescript
     verification: {
       yandex: "СКОПИРОВАННЫЙ_КОД",
     },
     ```
   - **Вариант В:** DNS-запись → добавить TXT-запись у регистратора домена
5. Подождать подтверждение (обычно 15 минут — 1 час)

**После подтверждения:**
6. Перейти: Индексирование → Карты сайтов
7. Добавить: `https://voloma.ru/sitemap.xml`
8. Проверить: Главная → Должен появиться значок «Индексируется»

**Ожидаемое время:** 1-3 дня до первой индексации

---

### 1.2 Google Search Console
**URL:** https://search.google.com/search-console

**Шаги:**
1. Зайти через Google-аккаунт
2. «Добавить ресурс» → выбрать «Ресурс с префиксом в URL»
3. Ввести: `https://voloma.ru`
4. Подтвердить права:
   - **Рекомендуемый:** HTML-тег → скопировать → добавить в `src/app/layout.tsx`:
     ```typescript
     verification: {
       google: "СКОПИРОВАННЫЙ_КОД",
     },
     ```
5. После подтверждения перейти: Карты сайта
6. Ввести: `sitemap.xml`
7. Нажать «Отправить»

**Ожидаемое время:** 3-7 дней до индексации

---

### 1.3 Bing Webmaster Tools (опционально, для полноты)
**URL:** https://www.bing.com/webmasters

**Шаги:**
1. Войти через Microsoft/Google аккаунт
2. Добавить сайт
3. Импортировать из Google Search Console (есть такая кнопка)
4. Или подтвердить через HTML-файл/DNS

## Этап 2: Настройка аналитики (1-2 день после релиза)

### 2.1 Яндекс.Метрика
**URL:** https://metrika.yandex.ru/

**Создание счётчика:**
1. «Добавить счётчик»
2. Заполнить:
   - **Имя:** Волома (voloma.ru)
   - **Адрес сайта:** `https://voloma.ru`
   - **Часовой пояс:** Москва (UTC+3)
3. Включить:
   - ✅ Вебвизор
   - ✅ Карта кликов
   - ✅ Аналитика форм
   - ✅ Отслеживание переходов
4. Скопировать **ID счётчика**
5. Создать файл `.env.local` на сервере:
   ```
   NEXT_PUBLIC_YANDEX_METRICA_ID=ВАШ_ID
   ```
6. Пересобрать и перезапустить сайт:
   ```bash
   npm run build
   pm2 restart voloma-site  # или ваш способ рестарта
   ```

**Настройка целей:**
В Метрике перейти: Настройки → Цели → Добавить цель

| Название цели | Тип | Идентификатор |
|---|---|---|
| Открыт конфигуратор | JavaScript-событие | `configurator_open` |
| Клик CTA (hero) | JavaScript-событие | `hero_cta_click` |
| Выбрана форма | JavaScript-событие | `option_shape_select` |
| Выбран размер | JavaScript-событие | `option_size_select` |
| Выбрана пропитка | JavaScript-событие | `option_color_select` |
| Выбрано качество | JavaScript-событие | `option_quality_select` |
| Просмотр изображения | JavaScript-событие | `gallery_image_view` |
| Клик по контакту | JavaScript-событие | `contact_click` |
| Индивидуальный заказ | JavaScript-событие | `custom_order_click` |
| Отправлена заявка | JavaScript-событие | `submit_request` |

**Настройка сегментов:**
- Новые vs вернувшиеся посетители
- Мобильный трафик
- География (Москва, Петрозаводск, другие города)

## Этап 4: Социальные сети и контент (2-4 неделя)

### 4.1 Создать/активировать соцсети
**VK (приоритет #1 для России):**
- Создать страницу бренда
- Заполнить: описание, сайт, контакты
- Добавить 3-5 постов с фото продукции
- Ссылка на voloma.ru в шапке

**Telegram:**
- Создать канал «Волома»
- Пост-закреп: о бренде + ссылка на сайт
- Контент: новинки, закулисье производства, отзывы

**Pinterest (для интерьерной тематики):**
- Создать бизнес-аккаунт
- Добавить 10-15 пинов с фото кашпо
- Каждая пина → ссылка на конфигуратор

---

### 4.2 Настроить Open Graph превью
**Проверить шаринг:**
1. **VK:** https://dev.vk.com/web/share/og-preview
2. **Telegram:** Отправить ссылку в избранное — проверить превью
3. **Facebook:** https://developers.facebook.com/tools/debug/

**Если превью не появляется:**
- Проверить `openGraph` в `layout.tsx`
- Убедиться, что изображение доступно по прямой ссылке
- Использовать инструмент отладки

---

### 4.3 Начать контент-маркетинг
**Блог на сайте (лучшее для SEO):**

**Темы для первых 5 статей:**
1. «Как выбрать кашпо для монстеры»
2. «5 идей для балконного озеленения»
3. «Почему карельская сосна — лучший материал»
4. «Как ухивать за деревянным кашпо»
5. «Тренды интерьерного озеленения 2026»

**Формат каждой статьи:**
- 1000-1500 слов
- 3-5 фото с alt-текстами
- Внутренние ссылки на конфигуратор
- JSON-LD Article schema

**Публикация:** 1 статья в 2 недели

## Этап 5: Мониторинг и оптимизация (1-3 месяц)

### 5.1 Еженедельные проверки
- **Яндекс.Метрика:**
  - Количество визитов
  - Источники трафика (поиск, прямой, соцсети)
  - Популярные страницы
  - Конверсия конфигуратора

- **Яндекс.Вебмастер:**
  - Ошибки индексации
  - Поисковые запросы (позиции)
  - Внешние ссылки

- **Google Search Console:**
  - Ошибки сканирования
  - Поисковые запросы
  - Мобильная пригодность

---

### 5.2 Раз в месяц
**Core Web Vitals:**
- https://pagespeed.web.dev/
- Цель: все показатели ≥ 90
- Если ниже — оптимизировать изображения/шрифты

**Проверка позиций:**
- Поисковые запросы в Яндекс.Вордстат: https://wordstat.yandex.ru/
- Отслеживать позиции по запросам:
  - "деревянные кашпо для интерьера"
  - "кашпо из дерева на заказ"
  - "кашпо карельская сосна"
  - "кашпо Петрозаводск"

**Анализ конкурентов:**
- Поисковая выдача по основным запросам
- Кто выше в выдаче? Что у них есть?
- Какие ключевые слова они используют?

---

### 5.3 Раз в квартал
**SEO-аудит:**
1. Проверить все страницы через Lighthouse
2. Проверить битые ссылки
3. Обновить sitemap.xml (если добавлялись страницы)
4. Проверить актуальность NAP (Name, Address, Phone) во всех справочниках
5. Проанализировать эффективность контента (какие статьи приносят трафик?)

**План на следующий квартал:**
- Какие новые ключевые слова добавить?
- Какие страницы создать?
- Какие справочники ещё не использованы?

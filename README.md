# Voloma Site

Сайт компании Voloma — деревянные кашпо для интерьера с конфигуратором для подбора формы, размера, пропитки и уровня качества.

## 🚀 Быстрый старт

### Требования
- Node.js 18+
- npm или pnpm

### Установка

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

### Сборка для продакшена

```bash
npm run build
npm start
```

## 🎯 Ключевые особенности

### Конфигуратор

Конфигуратор позволяет пользователям:
- Выбрать **форму** кашпо (узкое, квадратное, прямоугольное)
- Подобрать **размер** (S, M, L — только для узкого)
- Выбрать **пропитку** (без цветной обработки, под дуб, под палисандр)
- Определить **уровень качества** дерева (стандарт, премиум)
- Выбрать **с отделкой / без отделки** (влияет на цену)
- Добавить **колёсики** (входят в стоимость, кроме узкого S)

После конфигурации пользователь может отправить запрос менеджеру через Telegram, VK или MAX.

### Matching Strategies

Конфигуратор использует каскадную стратегию подбора вариантов:
1. **Exact match** — полное совпадение всех параметров
2. **Shape + Size + Finish** — совпадение без учёта качества
3. **Shape + Finish** — совпадение формы и пропитки
4. **Shape only** — только форма

Если точного варианта нет, показывается близкий вариант с пометкой.

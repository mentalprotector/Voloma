Название «Волома» отражает локальную идентичность и качество материала.

## Категории товаров
1. **Кашпо** — премиальные решения для террас, офисов и дома.
2. **Столы** — дизайнерские столы из массива карельской сосны.
3. **Погонаж** — качественные пиломатериалы и отделочные элементы.
